# ScaffoldOrganizer 사용자 가이드

![ScaffoldOrganizer Screenshot](docs/screenshot.svg)

브레인 덤프를 입력하면 AI가 구조화된 작업 표로 정리해 주고, Notion에 붙여넣기 쉬운 Markdown으로 저장할 수 있는 데스크톱 앱입니다.

---

## 1) 실행 방법

- **Windows**: `ScaffoldOrganizer.exe`를 더블 클릭
- **macOS**: `ScaffoldOrganizer.app`을 더블 클릭

처음 실행 시 `config.json`이 없으면 자동으로 생성됩니다. 위치는 OS별로 다음과 같습니다.

| OS | config.json 위치 |
|---|---|
| Windows | `%APPDATA%\ScaffoldOrganizer\config.json` |
| macOS | `~/Library/Application Support/ScaffoldOrganizer/config.json` |
| Linux | `~/.config/ScaffoldOrganizer/config.json` |

생성된 파일에서 `openai_api_key`와 `prompt_id`를 채운 뒤 앱을 재시작하세요.

---

## 2) 화면 구성

```
┌─ ScaffoldOrganizer ───────────── ─  ✕ ┐  ← (frameless 모드일 때)
├──────────────────────────────────────┤
│ [브레인 덤프 입력창]      [입력]     │
│                          [연속 입력] │
│ (최근 응답 요약)            (사용량) │
├──────────────────────────────────────┤
│ ToDo │ 대화 로그 │ Raw 결과 │ GUIDE  │
├──────────────────────────────────────┤
│ ┌──────────────────────────────────┐ │
│ │  🔢 │ 🧠 Task   │ 🛠 │ ⏱ │ ✅ │ 🧩 │ │
│ │  P0 │ ...       │... │.. │... │.. │ │
│ │  P1 │ ...       │... │.. │... │.. │ │
│ └──────────────────────────────────┘ │
│ [저장] [내보내기] [MD 복사] [새 세션] [불러오기] │
└──────────────────────────────────────┘
```

기본 창 크기는 **600 × 400 px** 입니다. 작업 표는 약 300px 높이로 표시되고, 컬럼 너비 합이 창 너비를 넘으면 가로 스크롤이 생깁니다.

---

## 3) 기본 사용 흐름

1. 입력창에 할 일이나 생각을 자유롭게 입력합니다.
2. **입력** 버튼을 누르면 AI가 분석해 작업 표를 채워 줍니다.
3. **ToDo 탭**에서 표를 직접 클릭해 편집할 수 있습니다.
4. **저장** 버튼으로 세션을 DB에 저장합니다.
5. **내보내기** 또는 **MD 복사**로 Notion에 붙여넣습니다.

---

## 4) 연속 입력 모드

- **꺼짐**: 입력할 때마다 즉시 전송
- **켜짐**: 여러 번 입력을 쌓았다가 마지막 줄에 `끝` 또는 `여기까지`를 적으면 한 번에 전송

---

## 5) 탭별 기능

| 탭 | 내용 |
|---|---|
| **ToDo** | 구조화된 작업 표. 셀 클릭으로 직접 편집 가능 |
| **대화 로그** | 최근 10개 대화 요약 확인 |
| **Raw 결과** | AI가 반환한 원문 Markdown + 최종 전송 원문 |
| **GUIDE** | 이 가이드 |

### ToDo 표 사용 팁

- **컬럼 너비**: Priority(`🔢`), Time(`⏱`), Status(`✅`)는 아이콘 너비로 작게, Task Description은 200px, Tool은 80px, Notes는 280px로 표시됩니다.
- **자동 줄바꿈**: Task / Tool / Notes 컬럼의 내용이 길면 셀 안에서 자동으로 줄바꿈되어 행 높이가 늘어납니다.
- **호버 툴팁**: 셀 위에 마우스를 올리면 잘려서 보이지 않는 긴 내용도 작은 팝업으로 전체 내용이 표시됩니다.
- **편집**: 어떤 셀이든 클릭하면 바로 입력 모드가 됩니다. ✅(완료) 컬럼은 체크박스로 토글합니다.

---

## 6) Markdown 내보내기 / 복사

- **내보내기**: `config.json`의 `export_dir` 경로에 파일로 저장합니다.
- **MD 복사**: 클립보드에 Markdown을 복사합니다. (Ctrl+V로 Notion에 바로 붙여넣기)
- AI 응답이 `🍎` / `🥑` 섹션을 포함하면 그 원문이 그대로 저장됩니다.

---

## 7) 세션 관리

- **새 세션**: 현재 상태를 초기화하고 새로 시작합니다.
- **불러오기**: 저장된 세션 목록에서 선택해 복원합니다.
- **대화 맥락 자동 복원**: 세션을 불러온 직후 입력창에 새 메시지를 보내면, 이전 대화 기록이 **자동으로 함께 전송**되어 AI가 맥락을 이어받아 응답합니다. 별도 작업 없이 그냥 입력만 하면 됩니다.

---

## 8) 설정 파일 (config.json)

수정 후에는 **앱을 재시작**해야 반영됩니다.

### 주요 항목

| 키 | 설명 |
|---|---|
| `openai_api_key` | OpenAI API 키 |
| `prompt_id` | OpenAI Prompt Asset ID |
| `prompt_variables` | 프롬프트 템플릿에 주입되는 변수들 |
| `default_model` | 사용 모델 (기본: `gpt-4.1`) |
| `runtime_overrides` | 프롬프트에 추가할 지시문 |
| `export_dir` | Markdown 저장 경로 (절대/상대 모두 가능) |
| `window_width` / `window_height` | 앱 창 크기 (px) |
| `native` | 독립 창 모드 (false면 브라우저로 열림) |
| `frameless` | OS 타이틀 바 제거 + 커스텀 타이틀 바 |

### `export_dir` 경로 형식 (Windows)

```json
"export_dir": "C:/Users/jiyulee/Documents/ScaffoldExports"
```
백슬래시(`\\`) 대신 슬래시(`/`)를 써도 정상 동작합니다.

### Frameless 모드

`"frameless": true`로 설정하면 OS의 두꺼운 타이틀 바가 사라지고 24px 짜리 슬레이트 색 커스텀 타이틀 바가 표시됩니다.

- **드래그**: 타이틀 바 영역을 잡고 창을 자유롭게 이동
- **─** : 최소화
- **✕** : 종료

## 9) 자주 묻는 질문

**Q. 작업 표가 비어 있어요.**  
A. Raw 결과 탭에서 응답에 표가 포함되는지 확인해 주세요.

**Q. Markdown이 JSON으로 저장돼요.**  
A. Raw 결과에 `🍎`/`🥑` 섹션이 포함되는지 확인해 주세요.

**Q. 설정을 바꿨는데 반영이 안 돼요.**  
A. 앱을 종료 후 다시 실행해야 합니다.

**Q. 이전 세션을 불러왔는데 맥락이 이어지나요?**  
A. 네. 세션 불러오기 후 첫 입력 시 이전 대화 전체가 자동 전송됩니다.

**Q. 창 크기를 바꾸고 싶어요.**  
A. `config.json`의 `window_width`와 `window_height`를 수정하고 재시작하세요.

**Q. 표 셀에 긴 글이 잘려서 보여요.**  
A. 셀에 마우스를 올리면 전체 내용이 툴팁으로 보입니다. 또한 셀 너비 안에서 자동 줄바꿈도 됩니다.

**Q. 표가 가로로 잘려요.**  
A. 표 영역에 가로 스크롤이 있습니다. 마우스 휠을 Shift와 함께 굴리거나 스크롤 바로 옮겨 보세요.

**Q. 닫기 버튼이 보이지 않아요 (frameless 모드).**  
A. 우측 상단의 ✕ 버튼을 누르세요. 드래그는 타이틀 바 빈 공간에서 가능합니다.
