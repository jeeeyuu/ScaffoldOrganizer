# ScaffoldOrganizer 사용자 가이드

브레인 덤프를 입력하고 구조화된 작업 표를 확인한 뒤, Notion에 붙여넣기 쉬운 Markdown으로 저장하는 앱입니다.

## 1) 실행 방법
- 배포된 앱(ScaffoldOrganizer)을 실행합니다.
- Windows: `ScaffoldOrganizer.exe`를 더블 클릭합니다.
- macOS: `ScaffoldOrganizer.app`을 더블 클릭합니다.

처음 실행 시 `config.json`이 없으면 자동으로 생성됩니다. 생성된 파일에서 `openai_api_key`와 `prompt_id`를 채운 뒤 재시작하세요.

## 2) 화면 구성

```
┌────────────────────────────────────┐
│ [브레인 덤프 입력창]    [입력]     │
│                         [연속 입력]│
│ (최근 응답 요약)          (사용량) │
├──────────────────────────────────┤
│ ToDo │ 대화 로그 │ Raw 결과 │ GUIDE│
├──────────────────────────────────┤
│ [작업 표]                         │
│ [저장] [내보내기] [MD 복사]        │
│                  [새 세션] [불러오기] │
└──────────────────────────────────┘
```

## 3) 기본 사용 흐름
1. 입력창에 할 일이나 생각을 자유롭게 입력합니다.
2. **입력** 버튼을 누르면 AI가 분석해 작업 표를 채워줍니다.
3. **ToDo 탭**에서 표를 확인하고 직접 편집할 수 있습니다.
4. 편집 후 **저장** 버튼으로 세션을 DB에 저장합니다.
5. **내보내기** 또는 **MD 복사**로 Notion에 붙여넣습니다.

## 4) 연속 입력 모드
- **꺼짐**: 입력할 때마다 즉시 전송합니다.
- **켜짐**: 여러 번 입력을 쌓았다가 한 번에 전송합니다.

연속 입력 모드에서는 마지막 줄에 `끝` 또는 `여기까지`를 입력하면 전송됩니다.

## 5) 탭별 기능

| 탭 | 내용 |
|---|---|
| **ToDo** | 구조화된 작업 표. 셀 클릭으로 직접 편집 가능 |
| **대화 로그** | 최근 10개 대화 요약 확인 |
| **Raw 결과** | AI가 반환한 원문 Markdown + 최종 전송 원문 |
| **GUIDE** | 이 가이드 |

## 6) Markdown 내보내기 / 복사
- **내보내기**: `config.json`의 `export_dir`에 파일로 저장합니다.
- **MD 복사**: 클립보드에 Markdown을 복사합니다.
- 모델 원문에 `🍎`/`🥑` 섹션이 있으면 그대로 저장됩니다.

## 7) 세션 관리
- **새 세션**: 현재 상태를 초기화하고 새로 시작합니다.
- **불러오기**: 저장된 세션 목록에서 선택해 복원합니다. 대화 맥락도 함께 복원됩니다.

## 8) 설정 파일 (config.json)

> 개발자는 `config_example.json`을 복사해 `config.json`을 만드세요.

주요 항목:

| 키 | 설명 |
|---|---|
| `openai_api_key` | OpenAI API 키 |
| `prompt_id` | OpenAI Prompt Asset ID |
| `prompt_variables` | 프롬프트 템플릿에 주입되는 변수들 |
| `default_model` | 사용 모델 (기본: `gpt-4.1`) |
| `runtime_overrides` | 프롬프트에 추가할 지시문 |
| `export_dir` | Markdown 저장 경로 |
| `window_width` / `window_height` | 앱 창 크기 (px) |

설정 변경 후에는 **앱을 재시작**해야 반영됩니다.

## 9) 자주 묻는 질문

**Q. 작업 표가 비어 있어요.**  
A. Raw 결과 탭에서 응답에 표가 포함되는지 확인해 주세요.

**Q. Markdown이 JSON으로 저장돼요.**  
A. Raw 결과에 `🍎`/`🥑` 섹션이 포함되는지 확인해 주세요.

**Q. 설정을 바꿨는데 반영이 안 돼요.**  
A. 앱을 종료 후 다시 실행해야 합니다.

**Q. 이전 세션을 불러왔는데 맥락이 이어지나요?**  
A. 네, 세션을 불러오면 대화 기록이 복원되어 이후 요청에 함께 반영됩니다.

**Q. 창 크기를 바꾸고 싶어요.**  
A. `config.json`의 `window_width`와 `window_height` 값을 수정하고 재시작하세요.
